Use Case Diagram

What is this diagram used for?

Illustrates how users interact with a system and what functions the system provides.

Class Diagram

What does this diagram represent?

Shows the static structure of a system, including classes, their attributes, operations, and relationships.

Activity Diagram

When is this diagram typically used?

Models the flow of control or data between activities, often used for business process modeling.

Sequence Diagram

What is the primary focus of this diagram?

Illustrates the order of messages exchanged between objects in a specific scenario over time.

State Machine Diagram

What does this diagram describe?

Shows the different states an object can have and the transitions between those states in response to events.

Component Diagram

What is the purpose of this diagram?

Depicts the components of a system and the relationships between them, often showing the physical packaging of elements.
